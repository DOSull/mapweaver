Welcome to MapTiler!

MapTiler™ lets you make tiled and woven maps, which can be used to present complex multivariate data, and... look cool doing it. However, it takes a while to load, which it is doing now in the background.


So while you wait, here's some explanation.


By tiling an area based thematic map—a classic choropleth—and assigning data from the underlying areas to the overlaid tiles, we provide a way for the map to visually present more than one data item to the map reader in each map area. Tiles in each position in the tiled pattern carry different attributes in the dataset.
